Slot messages difference between B2B and AFTN

Purpose of application

Application parse slot messages from text files from two different sources. First source is AFTN (Aeronautical Fixed Telecommunication Network), second source is generated by application that observe changes in flight parameters of flights in B2B and based on that infer slot messages. Goal is to compare how goodly or badly detection based on parameters change work compared to AFTN messages.
